## 📖 Original Description
As an Adviser, I want to view a library of published templates so that I can browse available content.


## 🎯 Acceptance Criteria (验收标准)
Given I am logged in as an Adviser
* * * * When I open the Templates screen
Then I see a list of templates displayed as cards
And only templates with status Published are shown
And each template card displays:
* * * * * Title
Description
Thumbnail/image
Tags
Actions (Preview, Use Template)
